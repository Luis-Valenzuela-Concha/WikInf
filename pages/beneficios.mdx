# Beneficios para estudiantes de informática
Ser estudiante de la Universidad de Concepción te ofrece una serie de beneficios diseñados para apoyarte en tu desarrollo académico, profesional y personal. Además de licencias de software que normalmente son de pago, muchas empresas y organizaciones proporcionan acceso gratuito a servicios valiosos, como servidores privados, herramientas de desarrollo y plataformas en la nube. Estos recursos te permitirán mejorar tus habilidades técnicas y aprovechar al máximo las oportunidades tanto dentro como fuera del aula. En esta sección se detallan todas las ventajas y servicios disponibles para ti como estudiante.

## GSuite UdeC

El Departamento de Informática entrega a cada alumno una cuenta Google con correo @inf.udec.cl y un Drive asociado con **almacenamiento ilimitado**.

Para activarlo:

1. Entra a google.com e inicia sesión
2. El correo va a ser tu correo normal pero @udec.cl va a ser @inf.udec.cl
3. La contraseña es tu numero de matricula sin dígito verificador
4. Creas una contraseña nueva


## GitHub Student Developer Pack

GitHub es una plataforma de alojamiento de repositorios Git que facilita la colaboración entre múltiples personas en proyectos de código. GitHub ofrece a los estudiantes el GitHub Student Developer Pack, un conjunto de herramientas y recursos proporcionados por distintas empresas. Entre los beneficios más destacados se incluyen:

* GitHub Copilot.
* $200 USD en créditos para servicios en DigitalOcean.
* $100 USD en créditos para servicios en Microsoft Azure
* Notion Pro.
* Diversas plataformas de educación en línea.
* Distintos dominios gratuitos por un año. 
* La mayoría de los IDE de Jetbrains

El resto de los items y las instrucciones para acceder al beneficio esta [aquí.](https://education.github.com/pack/)
